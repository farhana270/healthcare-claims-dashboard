# 🏥 Healthcare Claims Analysis Dashboard

##  Project Overview

This project analyzes healthcare claims data using SQL and Power BI to identify cost drivers, hospital performance, and patient trends.

---

##  Objectives

* Analyze healthcare costs across medical conditions
* Identify top revenue-generating hospitals and insurance providers
* Evaluate admission trends and length of stay
* Build an interactive dashboard for business insights

---

##  Tools & Technologies

* PostgreSQL (SQL)
* Power BI (DAX, Data Visualization)
* Excel/CSV Dataset

---

##  Data Model

* **Patients Table** → demographic details
* **Hospitals Table** → hospital & doctor info
* **Claims Table** → admissions, cost, LOS, readmission

---

##  Key Features

* KPI Cards: Revenue, Avg Cost, Readmission Rate, LOS
* Cost analysis by medical condition
* Monthly admission trends
* Top hospitals & insurance providers by revenue
* Interactive filters (Gender, Disease, Insurance)

---

##  Key Insights

* Arthritis, Obesity, and Diabetes show highest treatment costs (~25–26K)
* Top hospitals contribute significantly to total revenue
* Insurance providers like Cigna and Blue Cross drive major revenue
* Admissions peak mid-year (June–August)
* Avg length of stay ≈ 15.5 days
